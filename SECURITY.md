# Security Policy

## Supported Versions

The following versions of Nyanchi are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Nyanchi, please report it responsibly.

### How to Report

- **Discord:** Contact `@scxry` directly
- **GitHub:** Open a [private security advisory](https://github.com/REDTY12/Nyanchi/security/advisories/new)
- **Email:** Contact `scxry@scxry.xyz`

### What to Expect

- ⏱️ **Response time:** within 48 hours
- 🔍 **Investigation:** 1-7 days
- 🛠️ **Fix release:** ASAP after confirmation

### Please DO NOT

- ❌ Open public GitHub issues for security vulnerabilities
- ❌ Disclose the vulnerability publicly before a fix is released
- ❌ Exploit the vulnerability for malicious purposes

### Scope

This policy covers:
- The Nyanchi web configurator
- The Discord bot
- API endpoints
- Authentication & session handling

Thank you for helping keep Nyanchi safe! 🐾
